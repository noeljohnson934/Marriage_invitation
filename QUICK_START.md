🔧 COMMON CUSTOMIZATIONS
───────────────────────────────────────────────────────────────────────

CHANGE WEDDING DATE:
1. Open script.js
2. Find: const WEDDING_DATE = new Date('2026-05-24T11:30:00')
3. Change to your date: ('YYYY-MM-DDTHH:MM:00')
4. Save and upload

CHANGE COLORS:
1. Open style.css
2. Edit variables at top (line 7-20)
3. Example: --gold: #d4af37; change to your color hex
4. Save and upload

ADD YOUR PHOTO:
1. Put image in assets/ folder
2. Add to HTML: <img src="assets/your-photo.jpg" alt="description">
3. Style in CSS as needed

ADD MUSIC:
1. Edit background audio src in index.html
2. Use free music from: pixabay.com/music or freepik.com
3. Replace URL in audio tag

❓ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────

SITE NOT LOADING:
→ Wait 2-3 minutes after upload
→ Clear browser cache (Ctrl+Shift+Delete)
→ Verify repo name is [username].github.io

MAP NOT SHOWING:
→ Check internet connection
→ Google Maps needs internet

MUSIC NOT PLAYING:
→ Click music button to enable
→ Browser needs user interaction first

FILES NOT UPDATING:
→ Clear browser cache
→ Check files are properly uploaded
→ Try different browser
