## 🎨 CUSTOMIZATION GUIDE

### Quick Edits
**To change wedding details**, edit `index.html`:
- Search "Joy" → Change bride name
- Search "Noel" → Change groom name  
- Search "24 May 2026" → Change date
- Search "11:30 AM" → Change time
- Update family information sections
- Change venue and address

**To change colors**, edit `style.css`:
- Scroll to top (line 7-20)
- Find `:root { --cream: #faf6f1; ... }`
- Change color hex values
- Your site auto-updates!

**To add music**:
- Replace the audio URL in `index.html`
- Use free music from Pixabay or Freepik

**To add photos**:
- Put images in `assets/` folder
- Reference in HTML: `<img src="assets/photo.jpg">`
